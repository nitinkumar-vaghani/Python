# Complete List of Python Concepts You Should Learn.

Here are the concepts grouped logically from beginner to intermediate.

## 1. Core Python Foundations

* Installing Python
* Running Python scripts
* Variables
* Data types (int, float, str, bool)
* Type conversion
* Basic operations (math, string operations)
* User input
* Comments and documentation

## 2. Collections (Built-in Data Structures)

* Lists (indexing, slicing, methods)
* Tuples
* Sets
* Dictionaries
* List comprehensions
* Iteration over collections

## 3. Control Flow

* if / elif / else
* for loops
* while loops
* break, continue
* Enumerate and zip
* Range function

## 4. Functions

* Defining functions
* Arguments, keyword arguments
* Default parameters
* Return values
* Scope (local vs global)
* Lambda functions

## 5. Modules and Packages

* import
* Creating your own modules
* Using built-in modules
* Understanding the Python package structure
* Virtual environments (venv)
* Dependency management basics (pip, Poetry)

## 6. Error and Exception Handling

* try / except / finally
* Raising exceptions
* Custom exceptions

## 7. Object-Oriented Programming (OOP)

* Classes and objects
* Attributes and methods
* Constructors
* Inheritance
* Composition
* Encapsulation
* dunder methods (**str**, **repr**)

## 8. Type Hints and Static Typing

* Why type hints matter
* Annotating variables
* Function type hints
* Optional, Union, Literal
* Working with mypy

## 9. Working With Files

* Reading/writing text files
* Working with JSON
* Working with CSV

## 10. Decorators

* What a decorator is
* Writing simple decorators
* Understanding wrapper functions

## 11. Async Programming Basics

* sync vs async
* async/await
* event loop
* asyncio module
* running async tasks concurrently

## 12. External APIs and Networking

* requests (sync)
* httpx (sync + async)
* Parsing JSON responses

## 13. Data Libraries (Optional but Useful)

* NumPy basics (arrays, vectorization)
* pandas basics (optional, only if you plan to use it later)

## 14. Testing

* pytest basics
* Writing simple unit tests
* Fixtures
* Async tests

## 15. Code Quality Tools (before writing real apps)

* black
* flake8
* isort
* mypy
* pre-commit basics

These are the essential Python concepts you must master before entering the FastAPI world.
